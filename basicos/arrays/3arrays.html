Creo un array con varias frutas desordenadas.

var frutas = ["Melones", "Kiwis", "Mandarinas", "Melocotones", "Aguacates", "Fresas", "Tomates", "Yinyoles"];

Una vez tengo este array creado, cuando se pulse el botón, ordenaré las frutas mediante la orden sort() y se llamará a la función quitarboton() para eliminar tanto el texto como el botón.

function ordenarFrutas() {
	document.getElementById("frutas").innerHTML = frutas.sort();
	quitarboton();
}
			
function quitarboton() {
	var elem = document.getElementById("boton");
	var text = document.getElementById("texto");
	elem.parentNode.removeChild(elem);
	text.parentNode.removeChild(text);
}

Si pulsa el botón, este mensaje y el botón se autodestruirán al ordenar las frutas.
