Creo la variable fecha y le asigno valor mediante el metodo Date()

var fecha = new Date();

Una vez la variable fecha tiene una fecha, creo un array con los dias de la semana, siendo Domingo 0 y Sabado 6

var diafecha = new Array(7);
	diafecha[0] = "Domingo";
	diafecha[1] = "Lunes";
	diafecha[2] = "Martes";
	diafecha[3] = "Miercoles";
	diafecha[4] = "Jueves";
	diafecha[5] = "Viernes";
	diafecha[6] = "Sabado";

Cuando tengo este array creado, creo la variable diasemana, y le digo que para el array, obtenga el dia de la fecha y lo sustituya por el valor dado anteriormente.

var diasemana = diafecha[fecha.getDay()];

Después obtengo el parrafo diasemana, y le digo que me muestre el valor.

document.getElementById("diasemana") .innerHTML = diasemana;
